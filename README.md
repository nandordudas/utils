# @nandordudas Monorepo

This monorepo contains a collection of TypeScript utilities and tools.

## Packages

### [@nandordudas/utils](./packages/utils)

Type-safe utility functions for TypeScript projects. Features zero dependencies and robust error handling.

## Development

This project uses [pnpm](https://pnpm.io/) as its package manager.

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Build packages
pnpm build
```
